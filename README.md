# scaffold-default

## 目录结构

#### /mod - 模块

- /mod/index/index.js  index 脚本模块

- /mod/index/index.tpl.html  index 动态模块 pack时并入 main.js


#### /page - Ajax page

- /page/index.html - index 模块入口

#### /res - 全局资源

- /res/css/ - 全局样式(CSS,less)

- /res/images/ - 图片



#### main.entry.js - App 入口

## 编译

- npm server  启动dev server
- npm start 调试+监听文件
- npm build 编译生产文件
